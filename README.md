ideaVim, tmux, zsh

### install

```
source bootstrap.sh
```

phpstorm 安装 ideaVim 相关插件

tmux 安装tpm后 `<C-a>` + `I` install plugins

#### tmux

复制、分屏选择都采用vi模式

- `<C-a>` 指令前缀
- `-` 水平分屏
- `/` 垂直分屏
- `r` 重载配置
- `S` 保存session
- `R` 载入session
- ...

#### ideaVim

继承 [neovim配置](https://github.com/moonprism/nvim)
