### tmux

复制、分屏选择都采用vi模式

- `<C-a>` 指令前缀
- `-` 水平分屏
- `/` 垂直分屏
- `r` 重载配置
- `S` 保存session
- `R` 载入session

* xmodmap 换了键盘, 只能软件映射了（mac下使用karabiner-elements）。
